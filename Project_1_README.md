# AI Research Analyst (LLM + RAG System)

Designed and implemented an AI-powered research assistant that analyzes business reports using Retrieval-Augmented Generation (RAG).

Key capabilities:
- Extracts insights from unstructured PDF documents
- Reduces hallucinations using context-aware retrieval
- Compares LLM vs Prompt Engineering vs RAG performance

Tech Stack:
Python, OpenAI, ChromaDB, PyMuPDF, Token-based chunking

Key Highlights:
- Built end-to-end RAG pipeline
- Improved factual accuracy using vector search
- Identified limitations and future improvements toward Agentic AI systems

Next Phase:
Enhancing with multi-agent orchestration, evaluation metrics, and production-ready architecture.
